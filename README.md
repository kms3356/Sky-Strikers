# ✈️ Sky Strikers

웹 브라우저에서 바로 즐길 수 있는 1945 스타일의 클래식 비행 슈팅 게임입니다. 
복잡한 설치 과정 없이 HTML, CSS, JavaScript만으로 구현되어 언제 어디서든 가볍게 즐길 수 있습니다.

## 🎮 게임 플레이

아래 링크를 클릭하시면 깃허브 페이지를 통해 즉시 게임을 시작할 수 있습니다!

👉 **(https://kms3356.github.io/Sky-Strikers/)**

---

## 🛠️ 기술 스택
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Hosting:** GitHub Pages

## 💡 로컬에서 실행하는 방법
1. 이 레포지토리를 클론(Clone)하거나 ZIP 파일로 다운로드합니다.
2. 다운로드한 폴더를 열고 `index.html` 파일을 더블클릭하여 웹 브라우저로 실행합니다.
